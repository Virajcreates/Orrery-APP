# 3D Solar System Orrery

A beautiful, interactive 3D visualization of the Solar System built with **Three.js** and **Vite**.

## Features
-   **Real-time Orbits**: Accurate orbital mechanics using Keplerian elements.
-   **Interactive Camera**: Zoom, pan, and rotate to explore planets and comets.
-   **Comet Glow**: Custom procedural glow effects for comets like Halley and Encke.
-   **Date Control**: Time travel controls to view the solar system at any date.
-   **Responsive UI**: Sleek, modern interface for selecting celestial bodies.

## Technologies
-   **Three.js**: 3D Rendering.
-   **Vite**: Fast build tool.
-   **CSS2DRenderer**: Overlay labels.
-   **JavaScript (ES6+)**: Core logic.

## Setup
1.  Install dependencies:
    ```bash
    npm install
    ```
2.  Run development server:
    ```bash
    npm run dev
    ```
3.  Open `http://localhost:3000` in your browser.
